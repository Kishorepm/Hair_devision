# Handover Guides

Plain-English how-to guides for Raine — no developer needed. Each sub-project drops
its owner-facing guide here when it ships.

Planned (written as each sub-project lands):

- `website-editing.md` — how to change prices, hours, and photos on the new site.
- `booking-system.md` — managing the booking calendar, services, and reminders.
- `dashboard.md` — reading the KPI dashboard.
- `monthly-routine.md` — the 15-minute monthly admin + OKR check.

> Rule: if a guide needs code or a terminal to follow, it's not done. These are for
> a non-technical owner to run the business solo.
