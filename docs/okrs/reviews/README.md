# OKR Reviews

Cadence for keeping the [OKR tree](../okr-tree.md) honest.

- **Monthly (15 min):** update KR actuals, flag anything off-track, note one action.
- **Quarterly (45 min):** grade each KR 0.0–1.0, keep/drop/rewrite objectives for
  the next quarter, re-confirm baselines.

Copy the template below into a dated file, e.g. `2026-09-30-q3-review.md`.

---
