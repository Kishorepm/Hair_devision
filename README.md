# Hair Division

Standalone project repo for **Hair Division** — a hair salon at 446 Lake Road,
Takapuna, Auckland. Consulting engagement (Kish) to grow revenue and reduce the
owner's admin load.

> JD address: `24 Hair Division`. This is its own git repo, independent of the
> Nex_Doc monorepo.

## Where things live

| Path | What |
|------|------|
| `docs/specs/` | Design docs. Start with the engagement design. |
| `docs/okrs/` | The OKR tree and review check-ins. |
| `docs/strategy/` | Operating model, KPI definitions, positioning. |
| `docs/handover/` | Plain-English how-to guides for the owner. |
| `web/` | The Next.js website + CMS (sub-project 1, not built yet). |
| `tools/` | Dashboards & automations (sub-projects 3–4, not built yet). |

## Status

- **Backbone:** repo + OKR tree + structure — in progress.
- **Sub-projects (sequenced, website-first):** website rebuild → booking SaaS →
  admin dashboard → growth automations. Each gets its own spec before it's built.
